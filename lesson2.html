Практическое задание

  1. Почему код дает именно такие результаты?
var a = 1, b = 1, c, d;
  1. c = ++a; alert(c); // 2
    если инкремент перед переменной то сначало увеличит а потом выведет
  2. d = b++; alert(d); // 1
    если декремент перед переменной то сначало выведет а потом увеличит
  3. c = (2+ ++a); alert(c); // 5
    а в коде уже увеличилась на еденичку выше (2 + 3)
  4. d = (2+ b++); alert(d); // 4
    b выше увеличелось но здесь увеличется после операции
  5. alert(a); // 3
    в первом и третьем строке увеличилосо по единице
  6. alert(b); // 3
    во 2-ом и 4-ом строке увеличилосо по единице
